HOW TO USE 'Ryujinx' FOR SWITCH EMULATION: - Download Ryujinx from here: https://ryujinx.org/download - When completed, use your Switch Keys or get them from the #exclusive-content channel. Nintendo often takes down Key sources, so you probably won't find them online. - When downloaded, put your prod.keys file in the folder path: %appdata%\Ryujinx\system\ - Also make sure to install the firmware! You can download them from the SWITCH FIRMWARE folder in the Drive, then go to Ryujinx, then Tools, then press Install a Firmware from XCI or ZIP. and select the zip file! - How to install NSPs and XCIs: You don't have to install them! Just make sure they're in a folder, then go to Ryujinx, then in the top left, go to: Option -> Settings. Then press Add under Game Directories , select your folder with the games! - Have fun!
